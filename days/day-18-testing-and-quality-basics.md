# Day 18: Testing and Quality Basics

## Objective

Learn why testing supports confidence and safer change.

## Core Topics

- Manual testing
- Automated testing
- Assertions
- Validation
- Test notes
- Quality gates

## Lesson Content

Testing helps teams find problems earlier. Beginners can start with manual test steps and simple automated checks. In DevOps, tests often run automatically before code is merged or released.

## Hands-On Lab

1. Pick one Python script.
2. Add simple validation for user input.
3. Write manual test steps.
4. Record expected and actual results.
