# Day 2: Computer, Internet, and Developer Basics

## Objective

Build shared vocabulary for files, folders, applications, browsers, clients, servers, and APIs.

## Core Topics

- Files, folders, extensions, and paths
- Local computer vs remote server
- Browser, client, server, API, and database
- What developers mean by environment and dependency
- Why clear file organization matters

## Lesson Content

Most technical projects are organized collections of files. Some files contain code, some contain documentation, and some contain configuration. Beginners should learn to notice file names, extensions, and folder structure because these details matter in AI and DevOps projects.

Modern applications often involve a browser or app talking to a server through an API. That server may read or write data in a database or storage account.
