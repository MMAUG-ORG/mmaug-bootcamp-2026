# Day 11: Python Loops and Lists

## Objective

Use lists and loops to repeat work.

## Core Topics

- Lists
- `for` loops
- Repetition
- Iteration
- Automation thinking

## Lesson Content

Automation often means doing the same kind of task repeatedly without manually repeating every step. Loops are one of the first programming tools learners can use to think like automation builders.

## Hands-On Lab

1. Create `labs/day-11/study_checklist.py`.
2. Store seven study tasks in a list.
3. Use a loop to print them.
4. Add numbering if ready.

## Deliverable

- `labs/day-11/study_checklist.py`

## Mentor Notes

Connect loops to DevOps tasks such as checking files, reading logs, or validating environments.
