# Day 3: Command Line Fundamentals

## Objective

Use the terminal to navigate folders and create simple files.

## Core Topics

- Terminal and shell
- `pwd`, `ls`, `cd`, `mkdir`, `touch`, `cat`
- Relative and absolute paths
- Command history
- Reading errors without panic

## Lesson Content

The command line is a text-based way to control your computer. DevOps tools, cloud tools, Git, and Python all use command-line workflows. Beginners do not need to memorize every command, but they should become comfortable trying commands and reading output.

## Hands-On Lab

1. Open a terminal.
2. Navigate to your bootcamp folder.
3. Create `notes`, `labs`, and `project` if they do not exist.
4. Create `notes/day-03.md` from the terminal.
5. Add the commands you used to the notes file.

## Deliverable

- `notes/day-03.md`

## Mentor Notes

Let learners type commands themselves. Avoid taking over their keyboard unless truly necessary.
