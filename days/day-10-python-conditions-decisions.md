# Day 10: Python Conditions and Decisions

## Objective

Use conditional logic to make programs respond to different input.

## Core Topics

- `if`
- `elif`
- `else`
- Comparisons
- Basic input validation

## Lesson Content

Conditional logic lets programs make decisions. This is useful in automation scripts, chat workflows, validation tools, and command-line helpers.
