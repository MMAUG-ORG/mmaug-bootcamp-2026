# Day 23: Week 3 Review and Automation Challenge

## Objective

Connect cloud, DevOps, DevSecOps, AIOps, testing, and automation.

## Core Topics

- Cloud service categories
- DevOps lifecycle
- DevSecOps habits
- AIOps signals
- CI workflow failures
- Troubleshooting logs

## Lesson Content

Week 3 connects several ideas: cloud gives teams services, DevOps gives teams delivery practices, DevSecOps brings security earlier, and AIOps helps teams understand operational signals. Automation ties these ideas together.

## Hands-On Lab

1. Make a small change that causes your workflow to fail.
2. Read the GitHub Actions log.
3. Fix the problem.
4. Document the failure, cause, fix, and prevention.
