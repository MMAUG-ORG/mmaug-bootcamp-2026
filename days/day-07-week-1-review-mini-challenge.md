# Day 7: Week 1 Review and Mini Challenge

## Objective

Consolidate terminal, Git, GitHub, and Markdown basics.

## Core Topics

- Folder structure review
- Git status and commits
- GitHub repository review
- Markdown documentation
- Beginner troubleshooting

## Lesson Content

Week 1 built the learner's technical workspace. Before adding Python, AI, cloud, and DevOps concepts, every learner should confirm they can edit files, commit changes, push to GitHub, and explain their repository.

## Hands-On Lab

1. Create a new branch.
2. Add `notes/week-01-summary.md`.
3. Commit the change.
4. Merge the branch back into your main branch.
5. Push the result to GitHub.

## Deliverable

- `notes/week-01-summary.md`
