# Day 16: Azure Fundamentals for Beginners

## Objective

Map core Azure service categories to beginner AI and DevOps scenarios.

## Core Topics

- Resource groups
- Compute
- Storage
- Networking
- Microsoft Entra ID
- Monitoring
- Azure AI services

## Lesson Content

Azure groups cloud capabilities into services. Beginners should first learn the categories: where code runs, where data lives, how users and apps authenticate, how systems connect, and how teams monitor what is happening.

## Hands-On Lab

1. Choose a simple AI app idea.
2. Map the app to possible Azure service categories.
3. Explain what each category would do.
