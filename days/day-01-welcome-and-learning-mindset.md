# Day 1: Welcome, Goals, and Learning Mindset

## Objective

Understand the bootcamp purpose, the beginner journey, and how to track learning progress.

## Core Topics

- What AI means in everyday language
- What DevOps means in everyday language
- Why beginners need tooling habits before advanced projects
- Learning in public with GitHub
- Asking good technical questions

## Lesson Content

AI helps computers perform tasks that usually require human judgment, language understanding, pattern recognition, or decision support. DevOps is the practice of helping people, process, and automation work together so software can move from idea to useful outcome more reliably.

This bootcamp connects both areas. Learners will not become advanced AI engineers or DevOps engineers in 30 days. They will build the foundation needed to continue with confidence.

## Hands-On Lab

1. Create or confirm your GitHub account.
2. Create a local folder named `ai-devops-fundamentals-bootcamp`.
3. Create folders named `notes`, `labs`, `resources`, and `project`.
4. Copy the learner profile template into your repo.
5. Write your first learning goal.

## Deliverable

- `learner-profile.md`

## Mentor Notes

Keep this day calm and welcoming. Help learners understand that confusion is normal at the start.
