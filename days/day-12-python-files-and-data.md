# Day 12: Python Files and Data

## Objective

Read from and write to text files with Python.

## Core Topics

- Text files
- Reading files
- Writing files
- Data quality
- Notes as data

## Lesson Content

AI and DevOps both depend on data. AI systems use data to generate or retrieve useful output. DevOps systems use logs, configuration, and test results to understand what is happening.

## Hands-On Lab

1. Create `labs/day-12/note_writer.py`.
2. Ask the user for one thing learned today.
3. Save the note to a text file.
4. Read the file back and print it.

## Deliverable

- `labs/day-12/note_writer.py`

## Mentor Notes

Introduce the idea that files can become inputs for automation or AI workflows.
