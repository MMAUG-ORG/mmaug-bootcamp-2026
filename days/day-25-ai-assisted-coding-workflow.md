# Day 25: AI-Assisted Coding Workflow

## Objective

Use AI as a coding assistant while keeping human ownership of the work.

## Core Topics

- Asking for explanations
- Asking for test ideas
- Reviewing generated code
- Avoiding blind copy-paste
- Privacy and security caution
- Human accountability

## Lesson Content

AI assistants can help explain code, suggest improvements, and generate examples. The learner is still responsible for understanding, testing, and documenting the final work.

## Hands-On Lab

1. Choose one Python script from earlier labs.
2. Ask an AI assistant for one improvement.
3. Review the suggestion.
4. Apply only what you understand.
5. Document what you accepted, changed, or rejected.

## Deliverable

- `notes/day-25-ai-assisted-coding.md`

## Mentor Notes

Ask learners to explain each accepted change in their own words.
