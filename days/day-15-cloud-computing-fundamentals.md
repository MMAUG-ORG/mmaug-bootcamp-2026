# Day 15: Cloud Computing Fundamentals

## Objective

Understand why cloud platforms exist and what services they provide.

## Core Topics

- Servers
- Storage
- Networking
- Databases
- Public cloud
- IaaS, PaaS, and SaaS
- Regions and availability

## Lesson Content

Cloud platforms provide shared services that help teams build and run applications without owning all physical infrastructure. AI and DevOps projects often use cloud compute, storage, identity, monitoring, and automation services.

## Hands-On Lab

1. Describe a simple web app in Markdown.
2. Identify where compute, storage, network, and identity might fit.
3. Add a simple architecture diagram using bullets or Mermaid if ready.
