# Day 5: GitHub Foundations

## Objective

Publish the bootcamp repository to GitHub and understand basic collaboration features.

## Core Topics

- GitHub repositories
- README files
- Remote repositories
- Issues
- Pull requests
- Public portfolio basics

## Lesson Content

GitHub is where many software, AI, and DevOps projects are shared and reviewed. A clear README helps visitors understand what a repository is for. Issues help track work, questions, and improvements.

## Hands-On Lab

1. Create a GitHub repository.
2. Connect your local repo to GitHub.
3. Push your first commits.
4. Add a README with your goals and progress table.
5. Create one issue for a question you still have.

## Deliverable

- GitHub repository URL

## Mentor Notes

Expect authentication issues. Keep setup instructions patient and step-by-step.
