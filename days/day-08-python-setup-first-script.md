# Day 8: Python Setup and First Script

## Objective

Run Python locally and create a first beginner script.

## Core Topics

- Python interpreter
- Script files
- `print`
- Running Python from the terminal
- File extensions

## Lesson Content

Python is popular in AI, automation, data, cloud scripting, and DevOps. Beginners should start by running a very small script successfully before learning more language features.

## Hands-On Lab

1. Check your Python version.
2. Create `labs/day-08/hello_bootcamp.py`.
3. Print a welcome message.
4. Run the file from the terminal.
5. Commit the script.

## Deliverable

- `labs/day-08/hello_bootcamp.py`

## Mentor Notes

Help learners separate Python installation problems from Python language problems.
