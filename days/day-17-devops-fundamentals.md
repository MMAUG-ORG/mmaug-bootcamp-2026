# Day 17: DevOps Fundamentals

## Objective

Understand DevOps as culture, process, automation, and feedback loops, including where AI can safely assist delivery work.

## Core Topics

- Plan, code, build, test, release, deploy, operate, monitor
- Collaboration between development and operations
- CI and CD
- Small repeatable changes
- How DevSecOps and AIOps extend DevOps
- Agentic DevOps under human supervision
- AI-assisted planning, coding, testing, release notes, incident summaries, and operations review

## Lesson Content

DevOps helps teams deliver changes more reliably. It is not only tools. It includes collaboration, feedback loops, automation, and shared ownership. DevSecOps adds security into the delivery flow. AIOps uses AI and data to support operations work such as monitoring and troubleshooting.

Agentic DevOps means using AI assistants or agents to support delivery tasks such as breaking work into issues, drafting test cases, reviewing logs, summarizing incidents, generating release notes, or proposing workflow fixes. The important word is support. Humans still approve changes, protect secrets, check quality, and own production decisions.

## Mentor Notes

Help learners see DevOps as a lifecycle, not a job title only. For agentic DevOps, keep the emphasis on supervised assistance, traceability, and small reversible changes.
